# OMDB Movie Search

Merhaba Ben Mert Eşsiz, Yazılım Mühendisliği 4. sınıf öğrencisiyim. Bu proje i2i Systems staj başvurusu kapsamında geliştirdiğim bir film arama web uygulaması.

## Proje Hakkında

OMDB API'sini kullanarak film, dizi ve bölüm arayabileceğiniz tek sayfalık bir web uygulaması. Saf HTML, CSS ve JavaScript ile yazıldı — hiçbir framework kullanılmadı.

## Özellikler

-  Film, dizi ve bölüm arama
-  Türe göre filtreleme (Movie / Series / Episode)
-  Yıla göre filtreleme
-  Detaylı film bilgisi: oyuncular, yönetmen, IMDb puanı, özet
-  Son arama LocalStorage ile kaydedilir (sayfa yenilense bile kaybolmaz)
-  Sayfalama (Load More)
-  Tüm ekran boyutlarında çalışır (responsive)
